Markdown 是一款輕量型標記語言 (Markup Language)，只要使用簡單的符號排版文字內容，就能讓電腦軟體快速將符號轉換成網頁結構。

具體來說有 3 個好處:
- 用符號幫助快速排版內容，讓內容變得結構化
- [[notes/A-一次排版，多種形式的輸出]]
- 讓內容變得可快速瀏覽，節省讀者的時間

在網路上寫作，學習 Markdown 是不可或缺的一步。因為他能夠幫助你快速編排出有架構的內容，有感提升文章的好讀程度。
