date: "2022-10-31"
aliases: []
tags: Writing
---

使用 Markdown 寫作只需要一次排版，就可以輸出內容。

例如
- 簡報
	- 使用 IA Presenter
- 塗鴉筆記
	- h1 = title
	- h2 = subtitle
	- paragraph/bullet point = content
- PDF
	- Markdown 可直接渲染 (render) 出多種 PDF 排版格式
	- 寫作時不需要去管渲染出的效果，更換 CSS 就可以有不一樣的 PDF 格式了

一次排版，多種形式的輸出。
